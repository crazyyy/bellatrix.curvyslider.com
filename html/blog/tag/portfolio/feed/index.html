WordPress Portfolio Advanced Settings

Bellatrix Theme contains a very unique and powerful Portfolio element. Before using portfolio make sure you have added your item (some posts or default items or a different post type). If you haven't add any posts yet, click here to learn how to add default items.

Below here we will assume as you already added Default Items posts and you have categories all set up.

To add a portfolio, you need to add a page. For more details about adding pages read this

Adding Portfolio to Your Page

  • Go to your page to edit
  • Activate the Rock Page Builder if not activated automatically
  • Click on "Add Element" button
  • Choose "12 / 12" for Columns. (You can also choose any other columns as you like)
  • Click on "Portfolio" element icon
  • Now you have added the Portfolio Element to your layout.
  • Click on the "Portfolio" text or edit icon on the element, a popup will come up.
  • Select "defulatitems" for post type (You can also select different post types.)
  • Select the categories you want to display.
  • Fulfill the details as you like.
  • When you are done, click "Save Changes" in popup (a.k.a. Modal)
  • Click on "Save" button to save the page. If you do not know how to save Rock Page Builder content click here.

Add Default Item Categories

Before adding items, you should add item categories. To do this,

  • Login to your Wordpress and go to "Default Items > Item Categories".
  • Enter your category name
  • Enter your category title
  • Click on "Add New Category"

Add Default Items

To add a Default Item or any other post type follow these steps;

  • Login to your Wordpress and go to "Default Items > Add New".
  • Enter your title.
  • Enter Your Excerpt
  • Set your featured image
  • Choose a category (make sure you have selected a category)
  • Publish it.
